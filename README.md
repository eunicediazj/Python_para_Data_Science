# Python_para_Data_Science
Curso de Python para Data Science: trabajar con funciones, estructuras de datos y excepciones

Es un proyecto con un notebook que pertecene a Alura y ha sido desarrollado por la alumna Eunice Díaz 
como parte del curso Python para Data Science: trabajar con funciones, estructuras de datos y excepciones.
Contiene los ejercicios elaborados durante las lecciones, así como otro Notebook con los ejercicios 
propuestos para practicar lo aprendido en las lecciones del curso.
